StoryPulse Template
------------------
- Fully static HTML/CSS/JS
- Open index.html in browser to view
- Each feature is on its own page/section
- Pricing table: Free, $20, $49, $99
- Dashboard placeholders show visitor stats
- Colors and icons fully editable

Instructions for buyers:
1. Open index.html in a browser to preview the template.
2. Replace placeholder images and text with your own content.
3. CSS and JS are fully editable to customize styles and functionality.
4. Keep folder structure intact to ensure everything works correctly.
